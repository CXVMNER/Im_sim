Project made in engine Godot 4.6.1

### Running the latest version of the project:

1. Download the Godot engine from https://godotengine.org/download/

2. Download the project:

    - git clone https://github.com/CXVMNER/Im_sim.git

    - or "<> Code" > "Download zip", unzip

3. Import project to Godot

4. Run the project
